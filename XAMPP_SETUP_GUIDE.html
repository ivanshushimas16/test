Student Schedule Management System

Complete XAMPP Setup Guide

Overview

This is a complete Student Schedule Management System built with:

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP with XAMPP
  • Database: MySQL

The system includes three user roles:

  • Admin: Full system management, user management, activity logs, backups
  • Staff: Course and schedule management, student enrollment
  • Student: View personal schedule and grades

Prerequisites

Required Software

  • XAMPP (includes Apache, PHP, MySQL)
  • A web browser (Chrome, Firefox, Edge, Safari)
  • A text editor (optional, for advanced customization)

System Requirements

  • Windows 7 or higher, macOS 10.7 or higher, or Linux
  • At least 500MB free disk space
  • No special ports should be blocked
⚠ Important: Make sure ports 80 (Apache) and 3306 (MySQL) are available and not used by other applications.

Installation Steps

Step 1: Download and Install XAMPP

1. Go to https://www.apachefriends.org/
2. Download the latest XAMPP version for your operating system
3. Run the installer and follow the on-screen instructions
4. During installation, make sure to select:
  • Apache
  • MySQL
  • PHP
  • phpMyAdmin (recommended)

Step 2: Extract Project Files

1. Locate your XAMPP installation directory (usually C:\xampp on Windows or /Applications/XAMPP on Mac)
2. Navigate to the htdocs folder
3. Create a new folder named student-schedule
4. Extract all project files into the student-schedule folder
✓ Your project structure should look like:
htdocs/ ├── student-schedule/ │ ├── index.html │ ├── login.html │ ├── admin-dashboard.html │ ├── staff-dashboard.html │ ├── student-dashboard.html │ ├── api/ │ ├── js/ │ ├── css/ │ ├── config/ │ ├── includes/ │ └── scripts/

Database Setup

Step 1: Start XAMPP Services

1. Open XAMPP Control Panel
2. Click Start next to Apache
3. Click Start next to MySQL
4. Both should show as running (green indicator)

Step 2: Create Database

Option A: Using phpMyAdmin (Recommended)
  • Open your browser and go to http://localhost/phpmyadmin
  • Click "New" in the left sidebar
  • Create a new database named student_schedule_db
  • Select the database and go to the "SQL" tab
  • Open the file scripts/01_create_database.sql in a text editor
  • Copy all the SQL code and paste it into the SQL tab in phpMyAdmin
  • Click "Go" to execute
Option B: Using Setup Script (Automatic)
  • Make sure Apache and MySQL are running
  • In your browser, go to http://localhost/student-schedule/setup.html
  • Follow the on-screen setup wizard
  • The system will automatically create the database and tables

Step 3: Insert Sample Data

1. Open phpMyAdmin again at http://localhost/phpmyadmin
2. Select the student_schedule_db database
3. Go to the "SQL" tab
4. Open the file scripts/02_seed_data.sql in a text editor
5. Copy all the SQL code and paste it into phpMyAdmin
6. Click "Go" to execute
✓ Your database is now ready with sample data! The system includes 3 users for each role (admin, staff, student) for testing.

Troubleshooting

Apache Won't Start

Problem: Apache shows error or won't start in XAMPP Control Panel
Solution:
  • Check if port 80 is already in use by another application
  • Try stopping and restarting Apache
  • If using Windows, try running XAMPP Control Panel as Administrator
  • Check the Apache error logs in the XAMPP folder

MySQL Won't Start

Problem: MySQL shows error or won't start
Solution:
  • Check if port 3306 is already in use
  • Try stopping and restarting MySQL
  • Delete the ibdata1 file in the XAMPP data folder (this will delete all data)
  • Reinstall XAMPP if the issue persists

Cannot Access http://localhost/student-schedule/

Problem: Browser shows "Connection refused" or page not found
Solution:
  • Make sure Apache is running (green indicator in XAMPP Control Panel)
  • Check that the project files are in the correct folder: htdocs/student-schedule/
  • Try http://localhost/ to see if XAMPP is working at all
  • Clear browser cache and try again

Login Page Loads but Cannot Login

Problem: Login button doesn't work or shows error
Solution:
  • Make sure MySQL is running in XAMPP Control Panel
  • Verify the database was created: go to http://localhost/phpmyadmin
  • Check that student_schedule_db database exists
  • Verify that the database tables and sample data were inserted correctly
  • Check the PHP error logs in the XAMPP folder

Database Connection Error

Problem: "Database connection failed" error
Solution:
  • Verify MySQL is running
  • Check the database credentials in config/database.php
  • Default XAMPP MySQL credentials: Username = root, Password = (empty)
  • Try accessing phpMyAdmin to test the connection

Buttons Don't Work or Show No Response

Problem: Dashboard buttons don't respond to clicks
Solution:
  • Open browser Developer Tools (F12) and check the Console for errors
  • Check the Network tab to see if API calls are being made
  • Make sure JavaScript is enabled in your browser
  • Try a different browser to see if the issue persists

Need More Help?

For advanced troubleshooting:
  • Check browser console: Press F12 → Console tab
  • Check network calls: Press F12 → Network tab → try an action
  • Check PHP error logs in XAMPP folder
  • Test direct API calls: http://localhost/student-schedule/api/test-server.php
